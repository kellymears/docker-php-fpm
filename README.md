# Dockerized Bedrock

Bedrock in a box. Totally vanilla. Comes with `wp-cli`.

## Install

1. `git clone git@github.com:kellymears/docker-rock`

2. Set your .env variables _in the project root_.

3. Run `docker-compose up` in the project root.
